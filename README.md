# BILIBILI-Helper

一个用于管理 B 站关注主题并定时推送（邮件 + 微信）的控制台原型。当前版本重点提供：

- 关键词主题的增删改查
- 特别关注 / 付费 UP 主高亮配置
- 黑名单 / 白名单管理
- 推送频率与渠道配置
- 推送预览与关键词聚合建议展示

## 本地启动（uv）

```bash
uv sync
uv run python app.py
```

打开浏览器访问 `http://localhost:5000`。
